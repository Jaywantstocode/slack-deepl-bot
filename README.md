# slack-deepl-bot
A slack bot to translate your message to a different language.

currently supporting 10 languages.

"DE" - German
"EN" - English
"FR" - French
"IT" - Italian
"JA" - Japanese
"ES" - Spanish
"NL" - Dutch
"PL" - Polish
"RU" - Russian
"ZH" - Chinese


## requirements
Heroku

Slack

DeepL account
